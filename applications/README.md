# applications/

Одна папка на вакансию: `applications/<company-short>-<role-short>/`

| Файл | Назначение |
|------|------------|
| `vacancy.md` | Текст/заметки по вакансии (для агента) |
| `cover.md` | Сопроводительное (3–5 предложений) |
| `cv.md` | Адаптированное резюме — только для **не-hh** площадок |
| `*.pdf` | Генерируется скриптом, в git не кладётся |

**Slug:** латиница, дефисы, без пробелов. В чате и путях используйте прямые слэши: `applications/acme-pm` (и на Windows).

**Git:** ваши папки с откликами в `.gitignore` — в репозиторий не попадут, только локально.

## Папки с `_`

Каталоги вроде [`_example-do-not-track/`](_example-do-not-track/) — **образец структуры**. Не добавляйте их в [tracker.md](../tracker.md). Для реального отклика скопируйте структуру в новую папку, например `applications/acme-pm/`.
