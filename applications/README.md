# applications/

One folder per vacancy: `applications/<company-short>-<role-short>/`

| File | Purpose |
|------|---------|
| `vacancy.md` | Job notes for the agent (URL, fit, gaps) |
| `cover.md` | Cover letter (3–5 sentences) |
| `cv.md` | Tailored resume — **non-hh** platforms only |
| `*.pdf` | Generated by script; not committed to git |

**Slug:** lowercase Latin letters and hyphens, no spaces. Use forward slashes in paths and chat: `applications/acme-pm` (Windows too).

**Git:** your application folders are in `.gitignore` — they stay local only.

## Folders starting with `_`

Directories like [`_example-do-not-track/`](_example-do-not-track/) are **structure examples**. Do not add them to [tracker.md](../tracker.md). For a real application, copy the layout to a new folder, e.g. `applications/acme-pm/`.
